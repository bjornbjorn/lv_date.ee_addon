h1. Date type for Low Variables (ExpressionEngine 2.x)

This is a very basic date type for Low Variables which will enable the user to select a date using the standard EE jQuery UI date chooser.

To install just drop it into the /low_variables/types/ folder and activate it in the extension settings.
